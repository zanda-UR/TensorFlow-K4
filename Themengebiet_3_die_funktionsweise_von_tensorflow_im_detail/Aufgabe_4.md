## Aufgabe 4:

**Einschätzung einer Rückmeldung**

Ein kleines Nachhilfeinstitut möchte auswerten, ob Teilnehmende nach einer bestimmten Anzahl besuchter Vorbereitungstermine eine positive Rückmeldung zur Prüfungsvorbereitung geben. Dafür liegen sechs einfache Beobachtungen vor. Der Wert `1` bedeutet positive Rückmeldung, der Wert `0` bedeutet keine positive Rückmeldung.

Entwickle ein Modell, das anhand der Anzahl besuchter Vorbereitungstermine einschätzt, ob eine positive Rückmeldung wahrscheinlich ist.

Begründe zusätzlich kurz:

- warum du die Daten in dieser Form speicherst,
- warum eine Wahrscheinlichkeitsfunktion für diese Fragestellung passend ist,
- warum die Vorhersage am Ende gerundet wird,
- warum die Genauigkeit als einfache Auswertung hier sinnvoll ist.

| Besuchte Vorbereitungstermine | Positive Rückmeldung |
| ----------------------------- | -------------------- |
| 1                             | 0                    |
| 2                             | 0                    |
| 3                             | 0                    |
| 4                             | 1                    |
| 5                             | 1                    |
| 6                             | 1                    |
